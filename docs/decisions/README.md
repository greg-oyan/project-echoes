# Decision records

Record durable methodological, data, licensing, architecture, and experiment decisions here. Use one Markdown file per decision with a zero-padded number and short slug, for example `0001-source-manifest-field-names.md`.

Each record should include:

- Status and date
- Context
- Decision
- Alternatives considered
- Consequences
- Evidence or sources
- Conditions that would trigger reconsideration

Do not rewrite accepted decisions silently; supersede them with a new record.
